## コンストラクタ

```c++
(1) segtree<S, op, e> seg(int n)
(2) segtree<S, op, e> seg(vector<S> v)

//Range Min Query
int op(int a, int b) {
    return min(a, b);
}
int e() {
    return (int)(1e9);
}
segtree<int, op, e> seg(10);
```
